hexo部署个人博客

#通过nodejs下载hexo
npm install -g hexo-cli

创建博客目录并进入

mkdir my-hexo-blog && cd my-hexo-blog

初始化 Hexo 项目

hexo init

安装依赖

npm install

#配置 Hexo 部署到 GitHub
#编辑博客目录下的 _config.yml 文件，找到 deploy 部分并修改：
yaml
deploy:
type: git
repo: https://github.com/你的用户名/你的用户名.github.io.git
branch: main # 注意：如果是旧仓库可能是 master 分支

#安装部署插件
npm install hexo-deployer-git –save

清除旧的静态文件

hexo clean

生成新的静态文件

hexo generate

部署到 GitHub

hexo deploy

#可通过hexo new “文章标题”生成新文件 然后再执行 hexo c && hexo g && hexo d 来部署新文件
